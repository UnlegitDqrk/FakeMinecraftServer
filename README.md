# FakeMinecraftServer

Run a fake Minecraftserver.
ChatConverter by md_5: Copyright (c) 2012, md_5. All rights reserved.

**Features**:
- Colorable motd
- Colorable samples
- Servericon
- "Online-"players
- Maxplayers
- Set protocoltext
- Set protocolversion
- Switch engine (engine/json)
- Change the port (default: 25565)
